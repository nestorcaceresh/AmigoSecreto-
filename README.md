# Challenge Alura - Amigo Secreto 🎁  

¿Quieres organizar un Amigo Secreto de manera fácil y sin complicaciones? Esta aplicación es tu mejor aliada. Con una interfaz intuitiva y herramientas prácticas, podrás gestionar la lista de participantes y realizar el sorteo en solo unos pasos. Perfecta para cualquier celebración o evento especial.  

### ¿Qué funciones incluye? 🔧  

📝 **Agrega participantes**: Solo ingresa los nombres en el campo correspondiente y presiona "Añadir". ¡Así de fácil!  

⚠️ **Verificación automática**: Si intentas ingresar un nombre vacío o con caracteres no válidos, recibirás una alerta para corregirlo.  

📋 **Lista en tiempo real**: Los nombres se añaden y actualizan al instante, asegurando que nada se pierda.  

🎯 **Sorteo instantáneo**: Con solo un clic en "Sortear Amigo", el sistema seleccionará un nombre al azar y lo mostrará en pantalla.  

### ¿Por qué elegir esta app?  

✅ **Sencilla y accesible**: No necesitas conocimientos técnicos para usarla.  

🔄 **Rápida y precisa**: Evita repeticiones o errores en la lista.  

🎉 **Entretenida**: El sorteo aleatorio agrega emoción a cualquier reunión.  

Con esta aplicación, organizar un Amigo Secreto será más fácil y divertido que nunca. ¡Anímate a probarla! 🎊
